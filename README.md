# forkaem
fork for rest of life
## Задание
```bash
Найти различия и суть clone и fork.
Сделать форк проекта в новую локальную репозиторию. Расписать по шагам
```
## Понятие clone и fork
```bash
clone - это fork, но в visual code studio.
fork - это clone, но в github.
```

## Шаги
```bash
1. Нажимаем на нужный нам файл репозитория.
2. Нажимаем карандаш в правой части экрана.
3. Пишем что хотим.
4. Коммитим изменения.
5. Подтверждаем коммит.
6. Делаем ещё различные изменения в нашей локальной копии репозитория на Github при необходимости.
7. Нажимаем "Create pull request" и создаём пул запрос.
8. Подтверждаем пул запрос.
9. Ждём одобрения.
```
